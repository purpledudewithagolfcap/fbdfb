+Version: 1.1
+ This is a reminder that comments can be made by adding + to the beginning of a line.
+ 
+ To add a new word, create a new line starting with the word, in lower case, that you want MC to read out
+ and add a colon and then the sounds needed to read-out the word.
+ 
+ Known sounds : a,b,c,d,e,f,g,h,i,j,k,l,m,n,o,p,q,r,s,t,u,v,w,x,y,z,#,~,&,@,!,%,( , )
+ '#' is 'th', '~' is 'ch', '& is 'E', '@' is A, '!' is 'I' '%' is 'oo' (as in Loot), '(' is 'ow', and ')' is 'oh'  
+ Below is an example of how to add a words, and how to use the extra characters
